Instructions for Using Script Runner Pro

Follow these steps to download and use Script Runner Pro from the webstore:

1. Download Script Runner Pro

Visit the Webstore to download Script Runner Pro.

2. Use any of our scripts at this GitHub Repository

For all cheats and scripts, visit our GitHub Repository.

3. Using Script Runner Pro

Open Script Runner Pro in your browser and configure it according to the following instructions:

  • Open Script Runner Pro.
  • Click on the "Add Script" button to load your script.
  • Paste the script code into the provided field.
  • Save the script and execute it as needed.
