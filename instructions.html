How to Download and Use Script Runner Pro

  1. Go to the Script Runner Pro Webstore.
  2. Click on the "Download" or "Add to Browser" button to install Script Runner Pro.
  3. Once installed, open Script Runner Pro from your browser extensions.
  4. Click on the "Open Script" button within Script Runner Pro.
  5. Upload the downloaded script from the previous step or paste the script URL if applicable.
  6. Follow the on-screen instructions in Script Runner Pro to execute the script.
